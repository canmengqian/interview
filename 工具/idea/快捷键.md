## 查找和替换

- 整个项目替换，Ctrl+Shift+R
- 当前页面替换，Ctrl+R

## VIEW

- CTR+E :查看最近使用的文件
- CTR+SHIFT+E: 光标点过的位置
- CTR+D: 选择文件进行对比

- scientific mode :科学模式

## Navidate 导航

- CTR+N : 查找类
- CTR+SHIFT+N ： 查找文件
- CTR+G : 查找代码行
- 双击SHIFT :查找所有
- SHIFT+F2: 上一个代码告警
- ALT+ :arrow_down: :下一个方法
- ALT+ :arrow_up: : 上一个方法
- ALT+F1 : 文件结构,用于查看**类方法, 项目结构等** :white_check_mark:
- ctrl+shift+t : 生成测试类 :white_check_mark:
- CTR+H : 查看类继承关系
- CTR+SHIFT+H : 查看方法继承关系

## Code

- CTR+/ :行级注释
- CTR+SHIFT +/ :块级注释
- CTR+ALT+L: 代码格式化

- CTR+ALT+O: 优化导入包

## Analyze 代码分析

- Inspect Code : 检查代码
- infer null : 空值推断

## Run

- Show Running List : 展示运行中的列表
- CTR+SHIFT +F8: 查看断点

## Tools

- Start SSH Session : ssh连接
